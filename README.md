# ICS4U-Unit-4-Project
Project introduction / overview
My game features "Cloudy with a chance of meatballs"'s main character, Flint Lockwood in his attempts to gain points by catching the yummy meatballs and avoid losing points by not colliding with the rotten meatballs!
What the project does
How to run the program
The program is run when the "main.java" is the main file and its runned
Project goals or purpose
The purpose of this project is to start off with AI starter base code and add your own features to it and enhance it demonstrating skills and concepts learned in ICS4U.
Installation or setup instructions
Make sure to create a username and password which is used for registeration and saved in a file, and using those same values to return to the game later on.
